About the Project

This academic project combines machine learning and optimization techniques to classify skin lesions. It's an assisting tool — not a replacement for a dermatologist — and aims to encourage early consultation when abnormalities are suspected.

Designed for educational use, the platform demonstrates how integrated ML pipelines can support clinical workflows, emphasizing privacy, accuracy, and interpretability.

How it works
